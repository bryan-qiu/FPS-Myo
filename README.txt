How to Set-Up:

1. Open Google Chrome and go the website of any online fps game
2. Open "fpscontroller.lua" in a text editor and modify the reload and switch weapon variables to match the keys of the online game you chose
3. Run "Myo Connect.exe"
4. Add "fpscontroller.lua" to the script manager and make sure it is first on the list
5. Use the Wave Out motion to calibrate the Myo

How to Use:

- Move arm around to move gun around
- Finger Spread Motion to shoot gun
- Fist Motion to reload
- Fist Motion to reset where gun is pointing at
- First Motion and then rotate arm to reload
